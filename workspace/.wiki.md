# Project Summary
The project is a comprehensive management system tailored for charitable organizations, utilizing AI analytics to optimize fundraising strategies, enhance operational efficiency, and improve donor engagement. It features a suite of tools for task management, performance tracking, financial oversight, and real-time data management through Supabase, fostering transparency and accountability in charitable activities.

# Project Module Description
- **Dashboard**: Central hub for key metrics, tasks, and team activities with enhanced error handling and RTL support.
- **AI Insights**: Provides performance analysis and actionable recommendations.
- **Training Platform**: Offers skill development programs and certifications.
- **Gamification System**: Engages users through points and rewards.
- **Teams Management**: Organizes teams, donors, and project assignments.
- **Activity Log**: Logs user actions and system events.
- **File Upload**: Facilitates document sharing within tasks and projects.
- **Leaderboard**: Ranks employees based on performance.
- **Accounting**: Manages financial transactions and reporting.
- **Supabase Integration**: Connects to a real-time database for data management.
- **Notification System**: Sends real-time updates and alerts with enhanced error notification management.
- **Admin Permissions**: Manages user roles and permissions with newly added functions.
- **Profile Management**: Allows users to manage personal information and settings.
- **Settings**: Customizes application preferences.
- **Authentication System**: Manages user authentication and session control.
- **Task Management**: Features task assignments, execution dates, and progress monitoring with modern deletion confirmation dialogs.
- **Analytics**: Provides insights into project performance, financials, and team productivity.
- **Error Management**: Tracks and resolves error logs, integrated with enhanced logging functionality and a new interface.

# Technology Stack
- **Frontend**: React, TypeScript, Zustand, Recharts, Lucide React
- **Styling**: Tailwind CSS
- **Build Tools**: Vite, pnpm
- **Database**: Supabase

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
