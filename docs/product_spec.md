# برنامج قياس الأداء التسويقي للجمعيات الخيرية

## 1. الرؤية العامة
برنامج رقمي موحد يعمل على الويب وتطبيقات الهواتف الذكية (iOS/Android) لمتابعة أداء فرق التسويق في الجمعيات الخيرية داخل السعودية. يركز البرنامج على قياس الأفعال، تطوير السلوكيات اليومية، وتحويل تحقيق الأهداف إلى تجربة لعب محفزة تشبه شاشات التداول المالية.

## 2. الشرائح المستهدفة
- **المدير العام**: يمتلك لوحة قيادة متقدمة، إدارة الصلاحيات، إعداد الأهداف، متابعة الربحية، ومراقبة خطوات المشاريع الإلزامية.
- **المحاسب**: يدير النظام المحاسبي الكامل، يتابع المصروفات، العوائد، الربح الفعلي، ويدقق الداتا.
- **المشرف**: ينسق فرق التسويق، يراقب الأداء اليومي، يتابع المؤشرات، ويغذي الموظفين بالتوصيات.
- **الموظف**: ينفذ المهام التسويقية، يسجل الإيرادات، يتلقى التوصيات، يتفاعل مع التحديات والمكافآت.

## 3. المنصة التقنية
| عنصر | اختيار مقترح |
| --- | --- |
| إطار الواجهة | Flutter (تطبيق هجين للويب والموبايل) مع تصميم متجاوب |
| الواجهة الخلفية | NestJS (Node.js) أو Django Rest Framework لتعدد الحسابات وتكامل الذكاء الاصطناعي |
| قاعدة البيانات | PostgreSQL + Redis للتخزين المؤقت، مع دعم تخزين سحابي (AWS S3 أو Azure Blob) للملفات |
| الذكاء الاصطناعي | نماذج تحليل الأداء (AutoML أو Azure Cognitive Services) + محركات توصية |
| إدارة الهوية | Keycloak أو Auth0 لدعم تسجيل الدخول الأحادي (SSO) وصلاحيات متعددة المستويات |
| التكاملات | واجهات API مع متاجر الجمعيات، واتساب بزنس، أدوات البريد والرسائل |

## 4. تجربة المستخدم
1. **شاشة الدخول**: لا يسمح باستخدام أي جزء من النظام بدون حساب. دعم الدخول الثنائي (OTP / MFA).
2. **الصفحة الأولى (لوحة المتصدرين)**: جدول حي يبيّن ترتيب الموظفين حسب الأداء، بإضاءة شبيهة بشاشات الأسهم.
3. **لوحة تحكم المدير**: مؤشرات الأداء، خريطة حرارية للنشاط اليومي، مقارنة الأداء في الوقت الفعلي، جدولة الذكاء الاصطناعي، إعدادات الأهداف السنوية والشهرية.
4. **واجهات الموظفين**:
   - إدخال البيانات اليومية (إيرادات، مهام، مشاكل، ملفات مرفوعة).
   - توصيات بالذكاء الاصطناعي لكل مهمة.
   - نظام نقاط، مكافآت فورية، تحديات أسبوعية.
   - شاشة تقييم الأداء وتلخيص ذكي.

## 5. سيكولوجية الألوان وتجربة اللعب
- **الأخضر المزرق** لعرض التقدم الصحي والربحية.
- **البرتقالي** للتحديات والمكافآت الفورية.
- **الأحمر المخفف** للتنبيهات ومناطق التحسين.
- مؤثرات حركة مستوحاة من لوحات الأسهم (Ticker) تعزز الشعور بالتنافس.
- تطبيق مبادئ الإدمان الصحي على اللعب: أهداف قصيرة، تغذية راجعة فورية، مكافآت متدرجة، وسرد قصصي للنجاحات.

## 6. الهيكل الوظيفي والمزايا الرئيسية
### 6.1 حساب المدير (Admin)
- إدارة الحسابات (إنشاء، تعطيل، تعيين صلاحيات).
- إدخال المصروفات، حساب الربح الفعلي، إعدادات العائد التلقائية مع إمكانية الربط بشيت إكسل.
- إنشاء مشاريع، تحديد المستهدفات (شهرية/سنوية) لكل موظف.
- متابعة دليل الخطوات الإلزامية للمشاريع وتقدم المهام الإلزامية.
- استخراج الداتا (CSV/Excel)، التخزين السحابي، سجل العمليات.

### 6.2 حساب المحاسب
- نظام محاسبي لكل مشروع والإجمالي.
- تتبع الصرف والعائد، تحليل الربحية لكل موظف ولكل مشروع.
- تقارير مالية دورية، مراجعة المصروفات، إدارة الحوافز المالية.

### 6.3 حساب المشرف
- تعيين الموظفين للمشاريع، مراقبة التقدم، إرسال المهام والتنبيهات.
- تلقي تقارير AI عن نفسية الموظفين وتوصيات التعامل.
- لوحة مقارنة المؤشرات بين الموظفين، تقييم الأداء، تحديد نقاط القوة والضعف.

### 6.4 حساب الموظف
- تسجيل الإيرادات لكل مهمة، إضافة مهام جديدة، مشاركة المشاكل، رفع ملفات.
- استقبال التوصيات، المهام، التنبيهات، التحديات الأسبوعية، المكافآت.
- متابعة نظام النقاط، استبدال المكافآت، تلقي تقييمات لحظية.

## 7. وحدة إدارة المشاريع والمهام
- إنشاء المشاريع وتوزيع المهام على عدة موظفين.
- تصنيف المهام وفق الأولويات (رقم الأولوية، الموعد النهائي، الأثر المتوقع).
- تتبع التقدم، خريطة حرارية للنشاط، أتمتة المهام المتكررة.
- إدخال بيانات إيراد كل مهمة، تحليل الصرف والعائد، توصيات تحسين أسبوعية.

## 8. الذكاء الاصطناعي والتحليلات
- **تحليل الأداء**: خوارزميات لتحديد الاتجاهات، التوقعات للأسبوع القادم، واكتشاف فرص التحسين.
- **محرك توصيات**: اقتراح أفضل الممارسات، تدريب مستمر، وسيناريوهات للتعامل مع مانعي التبرع.
- **كشف الحالة النفسية**: تحليل سلوك الأداء لتكييف الرسائل التحفيزية.
- **مكتبة الأفعال الناجحة**: تُحدَّث تلقائيًا من الإنجازات اليومية.

## 9. نظام التدريب وتطوير المهارات
- برنامج تدريبي مستمر، حصص تبادل خبرات، جلسات أسبوعية لعرض النجاحات.
- مكتبة محتوى تعليمي (مهارات الإقناع، البيع الخيري، إدارة العلاقات، تحليل البيانات).
- دراسة حالات ناجحة وفاشلة، استضافة خبراء.

## 10. مخطط قاعدة البيانات (مقترح)
- `users`: بيانات المستخدم، الدور، الحالة، الصورة.
- `teams`: الفرق وربط الموظفين بالمشرفين.
- `projects`: بيانات المشروع، المستهدفات، دليل الخطوات الإلزامية.
- `tasks`: المهام، الأولوية، الحالة، الملفات، الإيراد.
- `revenues`: تسجيل الإيرادات لكل مهمة أو موظف.
- `expenses`: المصروفات، التصنيفات، الربح.
- `leaderboard`: نقاط الأداء، مستويات المكافآت.
- `ai_recommendations`: توصيات فردية، تحديث أسبوعي.
- `training_sessions`: البرامج التدريبية، الحضور، التقييم.
- `activity_logs`: سجل العمليات لكل حساب.

## 11. لوحة المؤشرات (UI Widgets)
- مؤشر رئيسي للهدف العام (100,000 ريال) مع تتبع لحظي.
- مخطط شبيه بشاشة الأسهم للموظفين (أعمدة متحركة، ألوان حية).
- خرائط حرارية للنشاط اليومي لكل موظف.
- بطاقات مقارنة فورية بين الفرق والمشاريع.

## 12. إدارة الحوافز واللعب
- نظام نقاط متعدد المستويات (الإكمال، تحقيق الأهداف، الابتكار).
- مكافآت فورية عند تحقيق أهداف فرعية (تنبيهات داخل التطبيق + بريد).
- تحديات أسبوعية بين الفرق (فريق 1 وفريق 2) مع لوحة تقدم.
- نقاط قابلة للاستبدال بمكافآت (قسائم، إجازات، دورات تدريبية).

## 13. تكامل مهام الفرق
### فريق 1 (3 موظفين)
- أتمتة إعادة الاستهداف كل جمعة بناءً على تصنيف المتبرعين.
- واجهات لإدارة تصنيفات المتبرعين (أعلى من 1000، 500-1000، 100-500، 30-100 ريال).
- أداة لإنشاء نشرات إبداعية لكل تصنيف + توليد ردود ذكية بالذكاء الاصطناعي.
- تكامل مع المتجر لتفعيل أرقام المتبرعين وتحديث حالات الواتساب والقناة.
- مكتبة قصص نجاح، حقيبة إلكترونية للمشاهير، إدارة تصميمات وصفحات الهبوط.

### فريق 2 (4 موظفين)
- لوحة بحث ومتابعة لـ 150 مشهور يوميًا، مع نظام إقناع وجدولة اجتماعات.
- حقيبة إلكترونية محدثة، تطوير استراتيجيات التعامل، وطرق الإقناع.
- متابعة إيراد كل مشهور، رابط المتجر، المستهدف، اسم المشهور.

## 14. الأمان والسياسات
- سياسات استخدام واضحة، موافقات على الشروط قبل الدخول.
- مراجعة دورية للصلاحيات والحسابات.
- سجل عمليات لكل حساب، القدرة على تفعيل/تعطيل الحسابات.
- نسخ احتياطي سحابي، تشفير البيانات الحساسة، امتثال لمعايير خصوصية المتبرعين.

## 15. خارطة الطريق التنفيذية
1. **التحليل التفصيلي**: ورشة عمل مع أصحاب المصلحة لتأكيد المتطلبات.
2. **تصميم تجربة المستخدم**: مخططات واجهات، هوية بصرية، نموذج gamification.
3. **البنية التقنية**: إعداد البنية السحابية، قواعد البيانات، التكامل مع مزودي الهوية.
4. **تطوير MVP**: وحدات الحسابات، إدارة المشاريع، تسجيل المهام، لوحة المتصدرين.
5. **الذكاء الاصطناعي**: دمج محرك التوصيات، التحليلات التنبؤية، تقييم الحالة النفسية.
6. **الاختبار والإطلاق**: تجارب قبول المستخدم، خطة تدريب، وثائق الاستخدام.
7. **التطوير المستمر**: تحسينات بناءً على البيانات، إضافة وحدات تدريب متقدمة.
